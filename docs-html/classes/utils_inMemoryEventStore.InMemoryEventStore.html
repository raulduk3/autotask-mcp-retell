    Class InMemoryEventStore

    Simple in-memory implementation of the EventStore interface for resumability This is primarily intended for examples and testing, not for production use where a persistent storage solution would be more appropriate.

    Based on MCP TypeScript SDK reference implementation: https://github.com/modelcontextprotocol/typescript-sdk/blob/main/src/examples/shared/inMemoryEventStore.ts
